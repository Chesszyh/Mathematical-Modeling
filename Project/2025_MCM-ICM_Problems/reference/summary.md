## Who wins

* **影响奖牌成功的因素：** 该论文确定了影响一个国家奥运表现的因素。这些因素包括：
  * 人口：由于拥有更大规模的潜在运动员群体，人口大国往往赢得更多奖牌。
  * 人均 GDP：较富裕的国家可以分配更多资源来训练和支持运动员，从而获得更好的表现。
  * 东道主优势：由于场馆熟悉度、裁判潜在的偏袒以及主场观众的支持等因素，东道主往往会赢得更多奖牌。
  * 政治因素：该论文提到了前苏联和东欧国家的成功，这可能是由于政府对运动员的支持和资助。

* **模型开发：** 该论文提出了一个模型，该模型考虑了人口、人均 GDP 和一个随机误差项来预测一个国家获得的奖牌份额。然后，该模型被扩展到包括诸如东道主优势和政治影响等因素。

* **局限性：** 该论文承认该模型并未考虑所有影响奖牌数的因素。一些局限性包括：
  * 并非所有国家都参加每个项目。
  * 团体项目与个人项目：团体项目的奖牌按一枚奖牌计算，即使它们涉及多名运动员。
  * 该模型没有考虑可能影响一个国家在特定运动项目中的表现的特定运动因素。

总而言之，该论文通过确定相关因素并提出基本模型框架，为你的数学建模提供了一个起点。你可以考虑纳入这些因素，并可能添加其他针对 2028 年奥运会背景的因素，以提高你的奖牌预测准确性。

1. **基于人口的简单模型（Population-based Model）：**

    * **基本假设：** 运动员天赋在全球人口中呈正态分布，因此一个国家拥有的奖牌数应与其占世界人口的比例成正比。
    * **公式表达：**
        * 奖牌份额（medalsharei）= 该国奖牌数（medalsi）/ 所有国家奖牌总数（∑j medalsj）
        * 人口份额（popsharei）= 该国人口（populationi）/ 所有参赛国人口总数（∑j populationj）
        * 理想情况下：medalsharei = popsharei
    * **模型修正：** 由于奥运会参赛名额并非完全按人口比例分配，且存在团体项目等因素，实际情况并非如此简单的线性关系。因此，模型引入了人口份额的对数（log-levels）进行修正，以更好地拟合数据。
    * **检验方法：**
        * **Probit模型：** 用于检验一个国家赢得至少一枚奖牌的概率是否与其人口份额相关。
        * **Tobit模型：** 用于估计奖牌份额与人口份额之间的关系。

2. **奖牌生产函数模型（Medal Production Function Model）：**

    * **基本思想：** 将赢得奥运奖牌视为一个“生产”过程，需要投入人力（人口）、物力（经济资源）和组织能力。
    * **公式表达：**
        * 奥运人才（Ti）= f(Ni, Yi, Ai)，其中Ni为人口，Yi为国民收入，Ai为组织能力。
        * 奖牌份额（Mi）= g(Ti)，即奖牌份额是人才的函数。
    * **具体形式：**
        * 采用柯布-道格拉斯生产函数（Cobb-Douglas production function）来描述人才的生产：Ti = Ai \* Nγi \* Yθi，其中γ和θ为参数。
        * 采用对数函数将相对人才转化为奖牌份额：Mi = ln(Ti) / ∑j ln(Tj)。
    * **模型扩展：**
        * 由于国民收入可以表示为人口和人均收入的乘积（Y = N \* (Y/N)），模型最终估计的形式为：Mit = C + α ln(Nit) + β ln((Y/N)it) + dt + νi + εit，其中α和β为参数，dt为时间虚拟变量，νi为国家随机效应，εit为误差项。
        * 为了考虑其他影响因素，模型还加入了以下虚拟变量：
            * 东道主虚拟变量（Hostit）：表示该国是否是当届奥运会的东道主。
            * 前苏联及卫星国虚拟变量（Sovietit）：表示该国是否属于前苏联或其卫星国。
            * 其他计划经济国家虚拟变量（Plannedit）：表示该国是否属于其他非苏联的计划经济国家。
        * 为了考虑奥运“投资”的持久性，模型还引入了滞后一期的奖牌份额（Mit-1）。

**总结：**

这篇文章从简单的基于人口的模型出发，逐步引入经济因素（人均GDP）、政治因素（前苏联及其他计划经济国家）和东道主优势，构建了一个更加完善的奖牌生产函数模型。该模型最终通过Tobit回归进行估计，并考虑了时间效应和国家随机效应，以及奥运“投资”的持久性。通过比较不同模型的拟合效果，作者证明了考虑多种因素的复杂模型能够更好地预测奥运奖牌数。

3. **模型构建：** 文章使用了两种模型：

        Mt = C + α1Nt + α2 (Yt / Nt) + α3P + α4Ht + ε

        Mt = At (Nt)γ (Yt)θ

        取对数后，并用常数C、社会主义虚拟变量P和主办国虚拟变量Ht代替lnAt，得到：

        lnMt = C + α1 lnNt + α2 lnYt + α3P + α4Ht + e

        为了解决多重共线性问题（lnNt和lnYt高度相关），作者使用了一个更严格的版本，假设规模报酬不变：

        Mt = At (Nt)1-θ(Yt)θ

        取对数并变换后得到：

        ln (Mt / Nt ) = C + α2 ln (Yt / Nt ) + α3P + α4Ht + e

## machine learning

这篇文章使用机器学习方法，特别是两阶段随机森林模型，来预测奥运会的奖牌数，并取得了优于传统方法的预测效果。以下是该文章对你的建模可能提供帮助的重要观点：

1. **研究目标：** 提高奥运奖牌数预测的准确性，为体育博彩公司、赞助商、媒体以及体育管理部门提供更可靠的参考。

2. **研究背景：** 传统的奥运奖牌预测方法主要使用统计模型，如OLS回归、泊松回归和Tobit回归等。然而，这些方法通常难以超越简单的“朴素预测”（即假设下一届奥运会的奖牌数与上一届相同）。

3. **研究方法：** 本文首次将两阶段随机森林模型应用于奥运奖牌预测：

    * **第一阶段（分类）：** 使用随机森林分类器预测一个国家是否会获得奖牌（二元分类问题：获奖/未获奖）。
    * **第二阶段（回归）：** 仅对第一阶段预测会获奖的国家，使用随机森林回归器预测其具体的奖牌数。

    这种两阶段方法类似于统计学中的Hurdle模型和Tobit模型，但利用了随机森林处理非线性关系和复杂交互作用的优势。

4. **模型优势：**

    * **超越朴素预测：** 该模型在2008、2012、2016和2020年奥运会的预测中，均优于朴素预测，这是以往研究中未曾实现的。
    * **提高预测精度：** 与其他研究（如Scelles et al., 2020）相比，预测精度提高了约20%。
    * **随机森林的优点：** 随机森林是一种集成学习方法，结合了多个决策树的预测结果，具有处理高维数据、捕捉非线性关系、不易过拟合等优点。

5. **使用的变量（特征）：**

    * **因变量：** 奥运奖牌数（取对数以减少偏斜）。
    * **自变量：**

**1. 核心假设：**

* 奥运奖牌的获得并非完全随机，而是受到一系列社会经济因素以及历史表现的影响。
* 将奖牌预测分解为两个阶段可以提高预测的准确性：首先预测是否获奖，然后预测获奖数量。

**2. 两阶段随机森林模型：**

* **第一阶段：分类（是否获奖）**
  * 使用随机森林分类器。
  * 目标：预测一个国家是否会在奥运会上获得任何奖牌（二元分类：是/否）。
  * 输入特征：包括人口、GDP、参赛运动员数量、疾病发病率和死亡率、主办国、政治体制、地理区域以及上一届奥运会奖牌数等。
  * 输出：一个国家是否会获得奖牌的概率。
* **第二阶段：回归（获奖数量）**
  * 使用随机森林回归器。
  * 目标：仅对第一阶段预测会获奖的国家，预测其具体的奖牌数量。
  * 输入特征：与第一阶段相同。
  * 输出：预测的奖牌数量。

**3. 随机森林的优势：**

* **集成学习：** 随机森林是一种集成学习方法，通过组合多个决策树的预测结果来提高整体预测性能。
* **处理非线性关系：** 随机森林能够有效地捕捉特征与目标变量之间的非线性关系，这在奥运奖牌预测中很重要，因为很多因素与奖牌数之间的关系并非简单的线性关系。
* **处理高维数据：** 随机森林可以处理包含大量特征的数据集。
* **鲁棒性：** 随机森林对异常值和噪声数据具有一定的鲁棒性。
* **不易过拟合：** 通过随机抽样和特征子集选择等方法，随机森林可以有效地避免过拟合问题。

**4. 特征工程：**

* **因变量转换：** 对奖牌数取对数，以减少数据的偏斜程度，提高模型的预测精度。
* **特征选择：** 选择了多个与奥运奖牌数相关的社会经济和历史因素作为模型的输入特征。
* **特征处理：**
  * **归一化GDP：** 将GDP转换为全球GDP份额，以反映国家在全球经济中的相对地位。
  * **人口数量对数化：** 对人口数量取对数，以缓解人口增长对奖牌数的边际效应递减。
  * **分类变量：** 将参赛运动员数量、疾病发病率和死亡率、主办国、政治体制和地理区域等变量转换为分类变量。
  * **上一届奥运会奖牌数：** 将上一届奥运会的奖牌数作为重要的历史特征加入模型。

**5. 数据预处理：**

* **数据映射：** 将不同数据源中的国家/地区名称进行统一映射，以匹配奥运代表队的信息。
* **插值/外推：** 使用线性插值和针对不同特征的合理外推方法填补缺失数据。对于疾病数据，采用常数外推，避免过度解释局部事件的影响；对于GDP和人口数据，采用线性外推，以保持数据趋势的平稳性。
* **区域基准：** 对于完全没有数据的国家，使用其所在区域的平均值作为基准。

**6. 模型训练、评估和比较：**

* **交叉验证：** 使用“最后区块交叉验证”方法，即使用较早年份的数据作为训练集，最近年份的数据作为验证集，以模拟实际预测场景，并避免时间序列数据中的偏差。
